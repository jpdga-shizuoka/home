プライバシー•ポリシー

はじめに

静岡県ディスクゴルフ協会 (以下, "当協会") は, 氏名, 住所, 電話番号といった個人を識別しうる情報 (以下 "個人情報") を適切に保護することが, 当協会活動の基本であるとともに, 社会的責務であると考えております.

ここでは当協会の個人情報に関する取り扱いについての基本方針をご紹介いたします.

個人情報とは

当協会が, 当ウェブサイトや 当協会が運用するFacebookページ (以下 "Facebookページ") を通じて, または大会開催の募集に際して, 提供を受ける住所, 氏名, 電話番号, メールアドレスなど, 人を識別できる情報または固有の情報をいいます.

個人情報の収集

当協会は, お問合せ, 大会参加募集やFacebookページを通じて, 個人情報を収集することがあります. 当協会による個人情報の収集は, あくまで自発的な提供によるものであり, 個人情報を提供された場合は, 当協会が本プライバシー•ポリシーに則って, 当該個人情報を利用することを許諾したものとします. ご同意いただけない場合には, いかなる個人情報も提供されないようお願いいたします.

なお当協会は, 法的手段によることなく個人情報を第三者に開示することはございません.

個人情報の利用目的について

ご提供いただいた個人情報は, 当協会が以下の目的で利用するものとします.

  • 当協会が主催する講習会や大会のお知らせ, および連絡
  • 当協会が主催する講習会や大会結果の発表
  • 当協会が主催する講習会や大会の記念品の発送
 リンクについて

当ウェブサイトは, いくつかの外部サイトへのリンクを含みますが, プライバシー情報を共有するものではありません. リンク先ウェブサイトにて行われる個人情報の収集に関しては, 当協会では一切責任を負えませんので, リンク先ウェブサイトの個人情報取扱内容を必ずご参照ください.

アクセスログ•履歴情報について

当ウェブサイトは, 当ウェブサイトとしてアクセスログ等のアクセス情報を保存するための (データベース等の) 記憶装置を有せず, よってアクセスログ•履歴情報は一切保有しません.

なお当ウェブサイトの運用は, GitHub社 が提供するウェブ•ホスティング•サービスである GitHub Pages を利用していますが, GitHub Pagesに関するプライバシー•ポリシーは, Github Pagesプライバシー•ポリシー をご参照ください.

Googleサービスの利用について

当協会は, 大会参加オンライン申し込み等のため, Google社が提供する Googleフォーム および Googleスプレッドシート サービスを利用することがあります. 提供いただいた申し込み等情報は, 当協会が所有するGoogleアカウントの元, 非公開とし適切にかつ厳重に管理いたします.

なおGoogleサービスに関するプライバシーポリシーは, Googleプライバシーポリシー をご参照ください.

アクセス解析サービスについて

当ウエブサイトは, 当ウェブサイトへのアクセス情報を得るため, アクセス解析サービスを利用しません.

保証および責任制限

静岡県ディスクゴルフ協会公式ウェブサイトは, ご利用者の責任において行われるものとします. 当ウェブサイトおよび当ウェブサイトにリンクが設定されている他のウェブサイトから取得された各種情報の利用によって生じたあらゆる損害に関して, 当協会は一切の責任を負いません.
